# Exp - 6 STUDENT ONLINE EXAM REGISTRATION SYSTEM

# AIM:
Design a system to allow students to register, take exams online, and view results, while instructors create exams and admin manages the system.

# SRS (Procedure):
1. Student registers and logs in.
2. Instructor logs in, creates exams, adds questions, and publishes them.
3. Student views available exams, takes exams, and submits answers.
4. Student views results.
5. Admin manages users, exams, roles, and generates reports.
